Test if the n<sup>th</sup> bit is set.
```
(x & (1 << n))
```


Set the n<sup>th</sup> bit.
```
x = x | (1 << n)
```


Unset the n<sup>th</sup> bit.
```
x = x & ~(1 << n)
```

Toggle the n<sup>th</sup> bit.
```
x = x ^ (1 << n)
```
