```txt
Multi-line text insert:
1. CTRL + V (select column)
2. SHIFT + i (insert)
3. type text
4. ESC
```

```txt
Multi-line text append:
1. CTRL + V (select lines)
2. $ (extend block to end of lines)
2. SHIFT + a (append)
3. type text
4. ESC
```
