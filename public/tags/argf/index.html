shuffle lines

Jan 13, 2012

Using Ruby’s special stream, ARGF, shuffling the lines in a file is trivial:

$ cat foo.txt | 
ruby -e "puts ARGF.readlines.collect.shuffle rescue Errno::EPIPE"
